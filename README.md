# GitHub Badges Generator

A professional, sleek, and high-speed GitHub badge generator designed to elevate your repository's aesthetics. Customize alignment, orientation, colors, and styles in real-time.



---

## Features

- **Live Customization:** Change styles (Flat, Plastic, For the Badge, Social) instantly.
- **Smart Alignment:** Choose between Horizontal or Vertical layouts with Left, Center, or Right alignment.
- **Full Color Control:** Use default brand colors from JSON or override with your own hex codes.
- **Logo Tinting:** Customize the icon color (logoColor) to match your theme.
- **One-Click Export:** Copy ready-to-use Markdown or HTML snippets.
- **Fully Responsive:** Beautiful UI that works perfectly on mobile and desktop.
- **State Reset:** A dedicated Reset Modal to clear your workspace instantly.
